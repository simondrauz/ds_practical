# Notebooks Directory

Use this folder for exploratory and reporting notebooks that accompany the centralized analysis.

## Current Notebooks
- `00_setup_validation.ipynb` — Verifies environment setup, dataset access, and trajdata functionality.
- `02_agent_centered_trajectory_metrics_analysis.ipynb` — Computes per-agent-type trajectory metrics and visualizations for vehicle, pedestrian, bicycle, and motorcycle agents.
- `03_scene_centered_trajectory_metrics_analysis.ipynb` — Breaks down scene-level agent distributions and densities by agent type.

## Planned Notebook Types
